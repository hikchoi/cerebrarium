# vim

## modes

- vim is a `modal` editor.
- different modes that serves different tasks.

- normal mode
    - Navigating and editing
    - we spend more time reading, navigating and changing code than actually insert new code.
    
- insert mode 
    - inserting text
- visual mode
    - selecting and editing text
